**Description**:
SDIP is an open-source, autonomous AI strategy analyst. It ingests unstructured market news and uses Large Language Models to extract, score, and aggregate structured business events (e.g., funding rounds, acquisitions) into a unified Next.js dashboard. 

Engineered with Python, FastAPI, and Next.js, the platform features a strict Provider Pattern to prevent LLM vendor lock-in, a custom Repository layer for database abstraction, and a two-pass deduplication algorithm to minimize API costs. Achieved >70% test coverage using a deterministic Mock-First CI/CD architecture.
