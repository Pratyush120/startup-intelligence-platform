# Strategic Decision Intelligence Platform (SDIP)

## Project Overview
The Strategic Decision Intelligence Platform (SDIP) is a premium executive decision workspace designed to help leaders understand the market in under 10 seconds. It processes unstructured business events and presents high-confidence strategic narratives and quantitative data through a calm, modern, and highly polished interface.

## Features
- **Executive Brief**: AI-generated narrative layer highlighting the most critical market movements and recommendations.
- **Metric Grid**: Real-time business health metrics with sparklines for top companies and the broader market.
- **Intelligence Timeline**: Event-driven tracking of funding, acquisitions, and strategic shifts across the industry.
- **AI Recommendation Center**: Actionable insights supported by AI analysis and confidence scores.
- **Dynamic Search & Explainability**: Deep dive into the underlying data directly from the executive interface.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Library**: React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend (Upcoming)**: FastAPI + Python Intelligence Engine

## Architecture
SDIP uses a decoupled architecture. The frontend is a static/SSR Next.js application that consumes structured intelligence generated by an asynchronous Python backend pipeline. The UI emphasizes performance, accessibility, and a premium aesthetic (inspired by Apple, Linear, and Palantir).

## Installation
```bash
git clone <repository>
cd frontend
npm install
```

## Environment Variables
Copy the `.env.example` file to `.env.local` and configure:
```bash
cp .env.example .env.local
```
Ensure all required variables (like `NEXT_PUBLIC_API_URL`) are set.

## Local Development
Start the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.

## Production Deployment
The application is optimized for deployment on Vercel:
1. Push your code to a GitHub repository.
2. Import the project in Vercel.
3. Configure environment variables in the Vercel dashboard.
4. Deploy.

The build command is `npm run build`.
