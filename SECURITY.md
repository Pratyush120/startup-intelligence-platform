# Security Policy

## Supported Versions

Currently, only the latest major release of SDIP receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of SDIP seriously. If you discover a security vulnerability within SDIP, please do not disclose it publicly.

Instead, please send an e-mail to our security team at [security@sdip.example.com].
We will respond to your report within 48 hours and work with you to triage the issue.

Please include the following information in your report:
- A description of the vulnerability.
- Steps to reproduce the vulnerability.
- Potential impact of the vulnerability.
- Any suggested mitigations.

We will provide a timeline for resolution and will notify you when a fix is available and published. We appreciate your efforts to responsibly disclose your findings, and we will make every effort to acknowledge your contributions.
