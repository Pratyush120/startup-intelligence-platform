# SDIP Roadmap

This document outlines the planned trajectory for the Startup Data Intelligence Platform. The roadmap is subject to change based on community feedback and emerging technologies.

## v1: The Foundation (Current)
- [x] Multi-source data ingestion (Crunchbase, LinkedIn, Twitter)
- [x] Basic scoring engine (Growth velocity, Founder capabilities)
- [x] Modern UI Dashboard (Next.js, dark mode)
- [x] Open-source release and documentation
- [x] LLM-powered summarization

## v2: Enhanced Intelligence (Target: Q4 2026)
- [ ] **Predictive Modeling**: Introduce ML models to forecast runway and time-to-next-round.
- [ ] **Custom Dashboards**: Allow users to create and save custom dashboard layouts.
- [ ] **Webhook Integrations**: Push notifications for major startup events (e.g., funding, leadership changes).
- [ ] **More Collectors**: Integrate with GitHub (for open-source traction), App Store/Play Store (for app downloads), and Product Hunt.
- [ ] **Data Export**: Export capabilities to CSV, PDF, and direct CRM integrations (Salesforce, HubSpot).

## v3: The Network Era (Target: Q2 2027)
- [ ] **Advanced Knowledge Graph**: Predictive edge generation in Neo4j to find "hidden" connections between founders and investors.
- [ ] **Collaborative Workspaces**: Multiplayer mode for venture teams to share notes and scorecards.
- [ ] **Automated Due Diligence**: Generate comprehensive due diligence reports spanning financial, technical, and market risks.
- [ ] **Browser Extension**: A Chrome extension to bring SDIP insights directly to LinkedIn and Crunchbase pages.

## Future AI Features
- **Agentic Workflows**: Allow users to deploy autonomous agents to actively monitor and research specific sectors (e.g., "Monitor AI hardware startups in Europe").
- **Conversational UI**: A natural language interface to query the database (e.g., "Show me all seed-stage fintechs with ex-Stripe founders").
- **Automated Outreach**: Draft personalized intro emails using the synthesized founder data.
